A library of Claude AI skills built from real marketing practice at **RichWorks** - a results-driven marketing agency focused on social media, content strategy, and AI-powered growth.

These are not generic tips scraped from the internet. They are the actual frameworks, workflows, and playbooks we use with clients - packaged as context-aware Claude skills that activate automatically when relevant.

## How it works

These are **context-aware skills** - not slash commands you have to remember. Start a conversation about hashtags and `hashtag-strategy` loads. Ask about a flopped reel and `reel-diagnosis` kicks in. Talk about writing a landing page and `landing-page-copy` activates.

Each skill carries:
- The core framework or methodology
- Verbatim prompts and templates (in Slovak where the original is Slovak)
- Cross-references to related skills
- Concrete examples from real campaigns

## Language note

Most skills default to **Slovak output** - these were built for a Slovak-speaking audience. The frameworks and methodologies are universal; switch to any language by specifying it in your conversation.
