# Profile Growth (1000 followers a month, by the numbers)

**Core belief to repeat to the user:** there is no growth hack for social media. The
closest thing to a hack is data. Make the user work with their statistics before giving
generic advice.

**Language:** produce output in the brand's language (default Slovak). Keep a plain,
direct, no-bullshit voice.

## Diagnose first: why a profile is not growing

There are only 2 reasons. Decide which one applies before recommending anything.

**Problem 1 - not enough people see the profile.**
Fix: increase profile traffic (visits).

**Problem 2 - the profile cannot convince enough of the people who see it.**
Fix: optimize the profile for conversion.

Always open the profile's statistics first and read the actual numbers (profile visits last
30 days, followers gained last 30 days). Reason from those.

## The follower-traffic rule

To gain ~1000 followers a month you need roughly **20,000 profile visits a month**. If
visits are far below that, the bottleneck is Problem 1 (traffic), not the profile.

### 9 ways to increase profile traffic
- Interact with other accounts.
- Use a few hashtags.
- Share other people's content.
- Collaborate with competitors (build collaborations, not rivalry).
- Use other social networks too.
- Post your content into Facebook groups as answers to questions.
- Support the competition and build collaborations with them.
- Keep the relationship with existing fans alive.
- Use paid advertising.
- Use video.

## The profile conversion-rate formula (memorize this)

```
(A / B) * 100 = C%
A = followers gained (last 30 days)
B = profile visits (last 30 days)
C = conversion rate of the profile, in %
```

**Ideal profile conversion rate: 10-20%.** If the profile is below that, the problem is
conversion (Problem 2), and the 8 optimizations below apply.

### 8 ways to raise the profile conversion rate
- Optimize the profile photo.
- Put a keyword in the name/title.
- Pick a relevant profile category.
- Choose a short, strong @handle.
- Optimize the highlights.
- Put a call to action in the BIO.
- Tidy up the main feed.
- Max out the feeds (reels, guides, effects, shop...).

(For a step-by-step on each of these, see the profile-optimization skill.)

## Bonus for advanced users: measure and report the conversion of ANYTHING

### Understand the marketing funnel first
The funnel is a hierarchy of metrics. At the top sit easy metrics with many people; the
deeper you go, the harder the metric is to achieve. "Harder" means harder for the person to
do. Asking someone to watch a video is easy. Asking them to buy is much harder.

Example customer journey (illustrative only, real journeys are never this clean):
1. Watches a video
2. Visits the profile
3. Follows
4. Clicks the website
5. Views the website
6. Subscribes to the newsletter
7. Buys

**The lower a metric sits in the hierarchy, the more valuable it is.**

### The general conversion formula
Conversion of anything is computed the same way: `(A / B) * 100 = C%`.

- **B** = the metric you want to track directly (the more valuable, lower metric): a web
  click, a newsletter sub, a sale - pick one, measured over 30 days.
- **A** = the less valuable metric that leads directly to B (one step above it in the funnel).

Worked example: in the last 30 days 100 people subscribed to the newsletter (the lower
metric) and 500 clicked the website (the metric one step above). So `(100 / 500) * 100 = 20%`
web-click-to-subscriber conversion. Always divide the deeper-funnel count by the
shallower-funnel count that leads to it, then * 100.

## How to use this skill
1. Ask for / read the 30-day stats: profile visits and followers gained.
2. Compute the profile conversion rate. Classify: traffic problem or conversion problem.
3. Give only the fixes for the actual bottleneck (don't dump both lists).
4. If the user wants to track a different goal (clicks, subs, sales), set up the funnel pair
   and the `(A/B)*100` report for it.
5. Close on the principle: no hacks, work the data.
