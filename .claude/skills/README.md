# RichWorks Skills

RichWorks marketing agency practices, frameworks, and methodologies - packaged as
Claude skills. Each skill covers a distinct niche so only the relevant one activates
per conversation, never the full set at once.

Install:
```bash
claude install github.com/rondoniec/richworks-skills
```

## Conventions (apply to every skill, current and future)
- **Voice:** plain, direct, no-bullshit.
- **Language:** default output language is Slovak unless the user works in another language.
  Slovak templates and formulas are preserved verbatim.
- **Niche separation:** one niche per skill; cross-link related skills by name rather than
  merging them.
- **No self-promotion:** skills contain only frameworks and advice. No plugs, no upsells,
  no calls to join anything.
